## 📌 Overview

The **Municipal Analytics Web Portal** is a responsive front-end dashboard built with **HTML, CSS, JavaScript, Chart.js, and Papa Parse**. It loads municipal service request data from a CSV file, calculates KPIs in real time, and presents interactive charts that help users explore operational performance.

The application demonstrates how a lightweight web interface can deliver meaningful analytics without requiring a backend database.

---

## 🎯 Project Objectives

- Build a responsive analytics dashboard
- Read and process CSV data dynamically
- Display key performance indicators (KPIs)
- Create interactive charts
- Filter dashboard data in real time
- Demonstrate front-end data visualization

---

## 📂 Dataset

The dashboard uses a cleaned municipal service request dataset containing over **10,000 records**.

Example fields include:

- Request ID
- Department
- Request Type
- Priority
- Status
- Resolution Days
- Estimated Cost
- Citizen Satisfaction
- Date Opened
- Date Closed

## ▶️ Running the Project

1. Clone the repository.
2. Open the project folder in Visual Studio Code.
3. Start **Live Server**.
4. Open `index.html`.
5. Explore the dashboard using the interactive filters.
